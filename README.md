# Unix/Linux システムプログラミング

名著『Unix/Linuxプログラミング理論と実践』のコードによる内部構造の検証記録です。
書籍中の課題、プロジェクトにも挑戦中。
「OSの裏側で何が起きているのか」を、C言語のソースコードを通して探求しています。

**※ 読了まで毎日更新中**

##  学習記録と環境構成
- **[UNDERSTANDING_OS.md](./UNDERSTANDING_OS.md)**：OSの役割、リソース管理、Unix哲学に関する考察まとめ
- **[ENVIRONMENT.md](./ENVIRONMENT.md)**：コンパイル環境および検証用OSの構成詳細

## 構成と取り組み

- **理論の言語化と検証**
  `UNDERSTANDING_OS.md` にてOSの基本原則（カーネル空間、プロセス管理等）を整理し、コードによる実証を繰り返しています。
- **ビルド環境の整備**
  各ディレクトリで Makefile を構成し、効率的にコンパイルとテストが行える環境を構築。
- **バッファリングによる最適化**
  共通ライブラリ（utmplib）にて、ユーザー空間でのバッファリングによるシステムコール削減を実装。
- **ファイルシステムの解析**
  iノード、stat構造体、再帰アルゴリズムを用い、標準コマンドの挙動を低レイヤーから再現。

## ディレクトリ構造

- more/    : 端末ページャの基本実装
- who/     : ログイン管理とバッファリング処理
- ls/      : stat構造体とビットマスクによる属性解析
- pwd/     : iノードと再帰によるパス解決
- chap02ex/: 第2章 演習問題（cat, tail等の再実装）
- common/  : プロジェクト共通ライブラリ
